Añadir un nuevo elemento al HTML

  1. Creamos un elemento con createElement()
  2. Creamos el texto con createTextNode()
  3. Agregar el texto al elemento con appendChild(node)
  4. Agregamos el nuevo elmento a un nodo existente con appendChild(node)
  5. Con el primer botón añadimos el nuevo elemento como el último hijo del padre
  6. Si quieres colocarlo entre varios elementos utiliza el método insertBefore()
  7. También podemos añadirle un atributo al nuevo elemento con el método setAttribute()
